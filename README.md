# Netflix UI Clone

Its a responsive, pixel-perfect frontend replica of the Netflix landing page interface. 
This project demonstrates strong foundations in semantic HTML5 layout structures, modern CSS styling methodologies, 
and cross-browser responsive design.

## 🚀 Live Demo

[👉 Click here to view the live deployment](https://memodd.github.io/My-Portfolio-Project-1-UI-Clone/)]

## 🛠️ Tech Stack

*   **HTML5:** Semantic elements for clean structure and accessibility.
*   **CSS3:** Custom properties (CSS variables), Flexbox, CSS Grid, and media queries for fluid responsive layouts.

## ✨ Core Features & UI Focus

*   **Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens.
*   **Accordion Elements:** Structured interactive FAQ section designed for seamless JavaScript integration.
*   **Pixel-Perfect Accuracy:** Matches the exact branding, spacing, typography, and styling of the official
*   Netflix user experience.
*   **Reusability:** Most elements are made up of reusable classes making it easier to maintain code .
